Hell Jump for iOS

Install Hell Jump on your iPhone or iPad — no App Store required. Choose the method that works for your device.

SideStore LiveContainer TrollStore AltStore
⚠️ Important: Hell Jump for iOS is distributed as a .ipa file and is not available on the official App Store. You'll need one of the sideloading methods below. None of these methods require a jailbreak (except TrollStore on some versions). Choose based on your iOS version and what you have available.

SideStore

The easiest and most user-friendly option. Add the AramCZGames repo directly to SideStore and install Hell Jump in one tap.

SideStore

Hell Jump on SideStore

✓ Pros

  • Easy one-tap install from repo
  • Automatic update notifications
  • No PC needed after initial setup
  • Works on non-jailbroken devices

✗ Cons

  • App expires every 7 days
  • Requires refreshing to stay active
  • Needs Apple ID (use a burner)
  • SideStore itself needs initial PC setup

How to Install

  1. Install and set up SideStore on your device if you haven't already.
  2. Open SideStore, go to Sources (the browse tab), then tap the + button.
  3. Paste the AramCZGames repo URL below and tap Add Source.
    4. https://aramczgames.github.io/iosrepo/
  4. Find Hell Jump in the source and tap Get to install.
  5. Remember to refresh the app in SideStore every 7 days to keep it working.

LiveContainer

Run Hell Jump inside LiveContainer — no signing, no 7-day limit. A great option if you already use LiveContainer.

LiveContainer

Hell Jump on LiveContainer

✓ Pros

  • No 7-day expiry — runs indefinitely
  • No re-signing needed
  • Works alongside other apps cleanly
  • Good performance in container

✗ Cons

  • LiveContainer itself needs to be sideloaded first
  • Some apps may have container compatibility issues
  • Slightly more complex initial setup
  • Container may limit some system features
You must have Sidestore already installed and running on your device before following these steps.

How to Install

  1. Make sure LiveContainer is installed and set up on your device.
  2. Download the Hell Jump .ipa file using the button below.
  3. In the share sheet, tap Open in LiveContainer, or open LiveContainer and import the .ipa from the Files app.
  4. Tap the Hell Jump entry in LiveContainer and press Run.
  5. This is optional but if you tap and hold Hell Jump click Add to home screen then App Clip. You can add Hell Jump to your home screen.

TrollStore

The most permanent install method — no expiry, no re-signing. Requires a compatible iOS version.

TrollStore

Hell Jump on TrollStore

✓ Pros

  • Permanent — never expires
  • No Apple ID required
  • Runs like a native app
  • No PC needed after TrollStore setup

✗ Cons

  • Only works on specific iOS versions
  • Not compatible with iOS 17+ on most devices
  • TrollStore itself can be tricky to install
  • May break on OS updates
TrollStore only works on iOS 14.0 – 16.6.1 and select other versions. Check the TrollStore compatibility list before proceeding.

How to Install

  1. Confirm your iOS version is supported by TrollStore.
  2. Install TrollStore on your device using the appropriate method for your iOS version (see TrollStore docs).
  3. Download the Hell Jump .ipa file using the button below.
  4. Open the .ipa with TrollStore — tap Install when prompted.
  5. Hell Jump will appear on your home screen as a permanent install.
After installing via TrollStore, Hell Jump will have full persistence — it won't disappear even after reboots or iOS updates (within supported versions).
