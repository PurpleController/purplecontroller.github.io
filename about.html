About Us

A random indie game studio from Czech Republic making games we'd want to play.

Who we are

AramCZGames is a solo indie game studio based in Czech Republic. It started as a personal project — one IT nerd who wanted to see if he could actually make a game from scratch. Turns out, you can. The studio is currently focused on its debut title, Hell Jump, a hellish platformer that's been in development since 2025.

There's no big team, no office, no investors. Just a passion for games and a lot of late nights figuring things out. Everything here is built independently, from the code to the art to the website you're reading right now.
